## FastAddressParseUtil

### 🚀 工具介绍

FastAddressParseUtil 是一款高精度地址解析工具，专注于将非结构化地址文本智能解析为结构化数据（省 / 市 / 区 / 详细地址）。

基于优化算法与规则引擎，在复杂地址场景下依然具备稳定表现，整体识别准确率最高可达 **97%**，达到行业领先水平。

👉 **工具定位：面向生产环境的地址解析基础组件**

适用于电商、物流、用户数据清洗、风控等多种业务场景。
